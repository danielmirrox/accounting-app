# Contributing to Accounting Application

Спасибо за интерес к проекту! 🙏

## Как помочь

1. **Найти баги** — открыть Issue с описанием
2. **Предложить фичу** — обсудить в Issues перед PR
3. **Улучшить код** — Pull Request с объяснением

## Процесс разработки

### Перед PR убедись:
- [ ] Код собирается без ошибок: `mvn clean package`
- [ ] Коммит-сообщение понятно
- [ ] Нет конфликтов с main

### Стиль кода
- Java 11+ features
- Используй Lombok где уместно
- Документируй сложную логику

## Вопросы?

Открой Discussion или Issue!
